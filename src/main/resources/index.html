API de conversión entre unidades de temperatura (°Celsius - °Fahrenheit)

Esta API tiene como fin hacer la conversión tanto de °Celsius a °Fahrenheit como de °Fahrenheit a °Celsius.

A continuación se presentan las diferentes funcionalidades del API, su forma de uso al conectarse de diferentes formas.

Funcionalidades:

El API de conversión entre las unidades de temperatura cuenta con dos funciones específicas que permiten realizar la conversión de °Celsius a °Fahrenheit y viceversa, a continuación la definición de cada una de estas funciones:

Conversión de Temperaturas en °Celsius a °Fahrenheit

Para realizar la converción de unidades de temperatura de grados Celsius a grados Fahrenheit la API implementa la siguiente fórmula:

°F = °C × 9/5 + 32

A continuación podrá ingresar un valor de la escala de temperatura Celsius, es decir, entre -273°C y 100°C para probar el funcionamiento del API

Conversión de Temperaturas en °Fahrenheit a °Celsius

Para realizar la converción de unidades de temperatura de grados Fahrenheit a grados Celsius la API implementa la siguiente fórmula:

°C = (°F - 32) × 5/9

A continuación podrá ingresar un valor de la escala de temperatura Fahrenheit, es decir, entre -460°F y 212°F para probar el funcionamiento del API
